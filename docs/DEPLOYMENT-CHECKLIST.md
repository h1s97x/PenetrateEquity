# 内网部署检查清单

使用此清单确保部署过程顺利完成。

## 部署前准备

### 环境检查
- [ ] Node.js >= 16.x 已安装
- [ ] npm >= 8.x 已安装
- [ ] 内网服务器已准备好
- [ ] Web 服务器软件已安装（Nginx/Apache/IIS）
- [ ] 防火墙规则已配置
- [ ] 域名或 IP 地址已确定

### 项目准备
- [ ] 项目代码已获取
- [ ] 依赖包已下载（如离线环境）
- [ ] 配置文件已准备（.env.production）
- [ ] 部署脚本已准备

## 构建阶段

### 依赖安装
- [ ] 运行 `npm install` 成功
- [ ] node_modules 文件夹已生成
- [ ] 无错误或警告信息

### 项目构建
- [ ] 运行 `npm run build` 成功
- [ ] dist 文件夹已生成
- [ ] dist 文件夹包含以下文件：
  - [ ] index.html
  - [ ] assets/ 文件夹
  - [ ] 其他静态资源

### 构建验证
- [ ] 检查 dist/index.html 文件大小正常
- [ ] 检查 dist/assets/ 包含 JS 和 CSS 文件
- [ ] 检查文件总大小合理（通常 < 10MB）

## 部署阶段

### 文件传输
- [ ] dist 文件夹已复制到服务器
- [ ] 文件权限已正确设置
- [ ] 文件所有者已正确设置

### Web 服务器配置

#### Nginx
- [ ] 配置文件已创建
- [ ] root 路径已正确设置
- [ ] location / 规则已配置（try_files）
- [ ] 静态资源缓存已配置
- [ ] Gzip 压缩已启用
- [ ] 配置语法检查通过（nginx -t）
- [ ] Nginx 已重启

#### Apache
- [ ] .htaccess 文件已创建
- [ ] mod_rewrite 模块已启用
- [ ] RewriteBase 已正确设置
- [ ] Apache 已重启

#### IIS
- [ ] web.config 文件已创建
- [ ] URL Rewrite 模块已安装
- [ ] 网站已在 IIS 中创建
- [ ] 应用程序池已配置
- [ ] 网站已启动

### 网络配置
- [ ] 防火墙端口已开放（80/443）
- [ ] DNS 记录已配置（如使用域名）
- [ ] 负载均衡已配置（如需要）
- [ ] SSL 证书已安装（如使用 HTTPS）
