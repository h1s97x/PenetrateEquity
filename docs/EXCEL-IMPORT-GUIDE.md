# Excel 导入使用指南

## 快速开始

### 1. 访问导入页面

在首页点击"导入 Excel 数据"按钮，或直接访问 `/#/excel-import`

### 2. 下载模板

点击"下载模板"按钮，获取 Excel 模板文件。模板包含：
- 示例数据
- 字段说明
- 正确的格式

### 3. 填写数据

在 Excel 中填写你的股权数据：

#### 必填字段

| 字段名 | 说明 | 示例 |
|--------|------|------|
| 公司名称 | 公司或被投资公司名称 | 京海控股集团 |
| 客户编号 | 公司的唯一标识 | C001 |
| 股东名称 | 股东名称 | 张三 / 李四投资 |
| 股东客户编号 | 股东的客户编号 | S001 / C002 |
| 股东类型 | 个人或企业 | 个人 / 企业 |
| 持股比例 | 持股百分比（数值格式） | 1 或 60 或 100 |
| 投资金额 | 投资金额（元） | 60000000 |

#### 可选字段

| 字段名 | 说明 | 示例 |
|--------|------|------|
| 统一社会信用代码 | 企业信用代码 | 91110000xxx |
| 股东出资排名 | 股东排序 | 1, 2, 3... |

### 4. 上传文件

- 点击上传区域或拖拽文件
- 支持 .xlsx 和 .xls 格式
- 文件大小不超过 10MB

### 5. 选择根节点

输入要查看的公司：
- 客户编号（推荐）
- 或公司名称

### 6. 查看图表

导入成功后，点击"查看股权穿透图"即可查看可视化结果。

## 数据关系说明

### 向上穿透（查看股东）

系统会查找所有"客户编号"等于目标公司的记录，这些记录的"股东"就是该公司的股东。

**示例：**
```
记录: { 公司名称: '京海控股', 客户编号: 'C001', 股东名称: '张三', 股东客户编号: 'S001' }
结果: 张三(S001) 是 京海控股(C001) 的股东
```

### 向下穿透（查看投资）

系统会查找所有"股东客户编号"等于目标公司客户编号的记录，这些记录的"公司"就是该公司投资的公司。

**示例：**
```
记录: { 公司名称: '京海科技', 客户编号: 'C003', 股东名称: '京海控股', 股东客户编号: 'C001' }
结果: 京海控股(C001) 投资了 京海科技(C003)
```

## 数据示例

### 完整示例

```
| 公司名称 | 客户编号 | 股东名称 | 股东客户编号 | 股东类型 | 持股比例 | 投资金额 | 股东出资排名 |
|---------|---------|---------|-------------|---------|---------|---------|------------|
| 京海控股 | C001 | 张三 | S001 | 个人 | 60% | 6000 | 1 |
| 京海控股 | C001 | 李四投资 | C002 | 企业 | 40% | 4000 | 2 |
| 京海科技 | C003 | 京海控股 | C001 | 企业 | 100% | 10000 | 1 |
| 李四投资 | C002 | 李四 | S002 | 个人 | 100% | 4000 | 1 |
```

这个数据会生成以下结构：

```
张三(60%) ─┐
           ├─ 京海控股(C001) ─ 京海科技(C003)
李四投资(40%) ─┘
    │
   李四(100%)
```

## 常见问题

### Q: 没有客户编号怎么办？

A: 客户编号是必需的，用于建立层级关系。你可以：
- 使用公司的统一社会信用代码作为客户编号
- 自己生成唯一的编号（如 C001, C002...）

### Q: 股东客户编号怎么填？

A: 
- 如果股东是企业，填写该企业的客户编号
- 如果股东是个人，可以使用个人编号（如 S001, S002...）

### Q: 持股比例格式有要求吗？

A: 支持多种格式：
- 60（数字）
- 60%（百分比）
- 0.6（小数）

系统会自动转换为统一格式。

### Q: 股东出资排名有什么用？

A: 用于控制股东在图表中的显示顺序，数字越小越靠前。如果不填，系统会按数据原始顺序显示。

### Q: 可以导入多层级结构吗？

A: 可以！只要数据中包含完整的"客户编号"和"股东客户编号"关系，系统会自动递归构建多层级结构。

### Q: 导入失败怎么办？

A: 检查以下几点：
1. 文件格式是否正确（.xlsx 或 .xls）
2. 必填字段是否都有值
3. 客户编号是否唯一
4. 字段名是否与模板一致

### Q: 可以修改字段名吗？

A: 目前字段名必须与模板一致。如果你的数据字段名不同，需要在 Excel 中修改列名。
